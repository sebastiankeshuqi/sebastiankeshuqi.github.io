乘法逆元（一）
Created 2020-07-21| Updated 2020-07-21|数论乘法逆元
Post View:

Reference: 【https://zhuanlan.zhihu.com/p/86561431】

背景

竞赛中会出现a/b（可以整除）的结果仍然非常大的现象，当我们要输出这样的答案时，好心的出题人就允许我们输出这个运算在mod p意义下的值。

也就是说，我们求一个数x，使得在mod p意义下$a/b=a\times x$

这就是解同余方程$bx\equiv 1(mod\quad p)$

证明

Assume $bk$ $mod$ $p=1$

Then

$\frac{a}{b}$ $mod$ $p=(\frac{a}{b}$ $mod$ $p)\cdot$$1$

$\frac{a}{b}$ $mod$ $p=(\frac{a}{b}$ $mod$ $p)\cdot$$(bk$ $mod$ $p)$

$\frac{a}{b}$ $mod$ $p=(\frac{a}{b}bk$ $mod$ $p)$

$\frac{a}{b}$ $mod$ $p=(ak$ $mod$ $p)$

求解方法

扩展欧几里得法

$ax\equiv1(mod$ $p)$可以转化为$ax+py=1,$ where $y\in\mathbb{Z}$

可以使用扩欧来求解这个线性方程，复杂度大约$O(log(n))$

证明

Bezout’s Lemma

Given any integers a and b, not both zero, there exist integers x0 and y0 such that $gcd(a,b)=ax_0+by_0$

Proof:

Let S:={$ax+by:x\in\mathbb{Z},y\in\mathbb{Z},ax+by>0$}

Let d be the smallest element in S. Then d=au+bv for some integers u and v

We show that d|a. Indeed, let q and r be such that a=qd+r and 0<r<d. Then

r=a-qd=a-q(au+bv)=(1-qu)a-qvb. Then 0 < r < d and r is in S, a contradiction.

So r must be 0.

wherece d|a. Similarly d|b.

Let c be any common divisor of a and b. Then we have c dividing au+bv.

This implies c|d and hence |c|$\leq$|d|.

Therefore, c$\leq$d, proving that d=gcd(a,b)

Corollary

Let a and b be integers, not both zero, and let d := gcd(a, b).
Then {ax+by: $x\in\mathbb{Z},y\in\mathbb{Z}$}={kgcd(a,b):$k\in\mathbb{Z}$}

Proof:

Because gcd(a,b)|ax and gcd(a,b)|by. Therefore gcd(a,b)|(ax+by).

Therefore ax+by=kgcd(a,b) for any x and y that are not both zero.

In other words, if the equation ax+by=m has any solution, then m must be divided by gcd(a,b)

Euclidean Algorithm

Given a<b,
$$
b=aq_1+r_1\
a=r_1q_2+r_2\
r_1=r_2q_3+r_3\
\dots\
r_{n-2}=r_{n-1}q_n+r_n\
r_{n-1}=r_nq_{n+1}
$$
Then gcd(a,b)=$r_n$

递归表示

eu(a,b):q1,r1

->eu(r1,a):q2,r2

…

->eu(r{n-1},r{n-2}):q{n},r{n}

->eu(r{n},r{n-1}):q{n+1},0

->eu(0,r{n}):END

Extended EA

递归表示

eu(a,b):q1,r1

->eu(r1,a):q2,r2

…

->eu(r{n-2},r{n-3}):q{n-1},r{n-1}

->eu(r{n-1},r{n-2}):q{n},r{n} Here we have gcd=r{n}=r{n-2}-r{n-1}q{n}

->eu(r{n},r{n-1}):q{n+1},0

->eu(r{n-1},r{n-2}):q{n},r{n} Here we have gcd=r{n}=r{n-2}-r{n-1}q{n}

->eu(r{n-2},r{n-3}):q{n-1},r{n-1} Here we have r{n-1}=r{n-3}-r{n-2}q{n-1}. So gcd=r{n}=r{n-2}-(r{n-3}-r{n-2}q{n-1})q{n}=(1-q{n-1}q{n})r{n-2}-q{n}r{n-3}

Then we discover the secret:

For eu(n{z},n{z-1}): if we have n{z}x+n{z-1}y=gcd

Then for eu(n{z-1},n{z-2}): we have n{z-1}x’+n{z-2}y’=gcd, where x’=y-x(n{z-2}//n{z-1}), y’=x

费马小定理

根据费马小定理，若p为素数，$a\in\mathbb{Z}^+$，且a、p互质，则有$a^{p-1}\equiv1(mod$ $p)$

这时发现上式和$ax\equiv1(mod$ $p)$很类似

所以$ax\equiv a^{p-1}(mod$ $p)$

化简得到$x\equiv a^{p-2}(mod$ $p)$

如果$a^{p-2}$用快速幂计算，时间复杂度和exgcd同级，为$O(log(n))$

逆元的线性递推式

某些问题需要你求不止一个数的逆元，这个时候每次都运行$log(n)$的算法的效率显得不那么高了（例如洛谷P3811，要求求区间 [1,𝑛][1,�] 的所有数的逆元）

证明

假设我们想求a在mod p下的逆元，可以这样写$p=aq+r$

因此$aq+r\equiv0(mod$ $p)$

移项可得$a\equiv-r\cdot inv(q)$ $(mod$ $p)$

两边同时取倒数，得出$inv(a)\equiv -q\cdot inv(r)$ $(mod$ $p)$

由于$q=\lfloor p/a\rfloor=p//a,r=p$ $mod$ $a$，代入上式得到$inv(a)\equiv-p//a\cdot inv(p$ $mod$ $a)$ $(mod$ $p)$

因为不确定$-p//q$是不是负数，就加一个p

得到最终的递推式$inv(a)\equiv(p-p//a)\cdot inv(p$ $mod$ $a)$ $(mod$ $p)$

在具体做的时候加入模运算就可以把$\equiv$改成$=$了

然后就可以实现$O(n)$递推了
