Bezout’s Lemma

Given any integers a and b, not both zero, there exist integers x0 and y0 such that gcd(a,b)=ax0+by0gcd(a,b)=ax_0+by_0gcd(a,b)=ax0​+by0​

Proof:

Let S:={ax+by:x∈Z,y∈Z,ax+by>0ax+by:x\in\mathbb{Z},y\in\mathbb{Z},ax+by>0ax+by:x∈Z,y∈Z,ax+by>0}

Let d be the smallest element in S. Then d=au+bv for some integers u and v

We show that d|a. Indeed, let q and r be such that a=qd+r and 0<r<d. Then

r=a-qd=a-q(au+bv)=(1-qu)a-qvb. Then 0 < r < d and r is in S, a contradiction.

wherece d|a. Similarly d|b.

Let c be any common divisor of a and b. Then we have c dividing au+bv.

This implies c|d and hence |c|≤\leq≤|d|.

Corollary

Let a and b be integers, not both zero, and let d := gcd(a, b).
Then {ax+by: x∈Z,y∈Zx\in\mathbb{Z},y\in\mathbb{Z}x∈Z,y∈Z}={kgcd(a,b):k∈Zk\in\mathbb{Z}k∈Z}
