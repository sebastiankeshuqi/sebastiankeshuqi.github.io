Weakest Link Pruning
Created 2020-05-05| Updated 2020-05-07|machine learning
Post View:

See the youtube video StatQuest

Category 1

指标

Tree Score

Definition

$$
Tree\ Score=SSR+\alpha T
$$

where

T is the Tree Complexity Penalty, which compensates for the differences in the number of leaves.(叶节点数量)

$\alpha$ is a tuning parameter that we finding using Cross Validation.(可变参数)

找一个Tree Score最低的subtree作为pruned的新树

Sample

![Screen Shot 2020-05-05 at 22.54.03](/Users/aronqi/Library/Application Support/typora-user-images/Screen Shot 2020-05-05 at 22.54.03.png)

Tuning parameter

具体说来就是讲testing data和training data整合为full data，利用full data训练出一棵recursion Tree，接着调节$\alpha$的值，从0开始增加，每次一个$\alpha$都能得出一棵tree score最小的subtree，直到最后所有儿子节点都被pruned掉了我们就停。根据这种方法再相应地对training data和testing data分别生成一群subtree，然后拿这些subtree去应对新的training data，和新testing data，看哪一棵subtree在SSR上表现最好(sum of residuals最小)

Category 2

指标

Determining node class label

我们使用决策树是用来处理分类问题的。对一棵决策树的叶子节点，我们需要确定它所属的类别。我们引入一个misclassifying cost: $C(i|j)$，作为错误分类的cost function，代表把一个实际属于第j类的样本错误分类到了第i类的代价，当然$C(j|j)=0$。

在分类不存在偏向的时候，可以用简单的函数
$$
C(i|j)=
\begin{cases}
1\quad i\not=j\
0\quad i=j
\end{cases}
$$
当分类存在偏向时，例如我们需要对感染疾病的人进行分类，我们当然不愿意放过每一个潜在感染者，因而会把放走感染者的代价调高一些

那么对于一个叶节点t，The class label j(t) of t is defined to be
$$
j^(t)=\mathop{argmin}{i}\sum{j}C(i|j)p(j|t)
$$
当C(i|j)是简单形式时，
$$
\sum_{j}C(i|j)p(j|t)=1-p(i|t)
\\mathop{argmin}{i}\sum{j}C(i|j)p(i|t)=\mathop{argmax}_{i}p(i|t)
$$
其中p(i)的定义为占比，$p(i)=\frac{N_i}{N}$，即标签为第i种的样本在所有样本中的占比，$p(i|t)$代表第i种样本在树节点t所含的样本空间内的数量占比。

Misclassification cost

Define the misclassification cost r(t) of node t by :
$$
r(t)=\mathop{min}{i}\sum{j}C(i|j)p(j|t)
$$
Let R(t)=r(t)p(t). Using this, define the misclassification cost R(T) of tree T by
$$
R(T)=\sum_{t\in\widetilde{T}}R(t)=\sum_{t\in\widetilde{T}}r(t)p(t)
$$
其中$\widetilde{T}$代表所有叶节点构成的集合。

Weakeast-link cutting & minimal cost-complexity pruning

Definition. Let $\alpha\geq0$. For any node t, not necessarily a terminal node, define $R_{\alpha}(t)=R(t)+\alpha$. And for a tree T, define $R_{\alpha}(T)=R(T)+\alpha|\widetilde{T}|$, where $|\widetilde{T}|$ is the number of terminal elements (leaf nodes).

用$R_{\alpha}(t)$ 作为表征misclassification cost的量，此时split节点不一定能使cost降低了，因为，即便split能降低$R(T)$，$\alpha|\widetilde{T}|$的值同时会增加（假设$\alpha>0$）
